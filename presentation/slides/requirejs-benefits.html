Why Require.js?

  • Self-contained libraries that don't create anything in the global scope
  • Our code can be organized in maintainable ways
  • Deploy bundles of (minified) code to the browser
    • overhead in opening and closing requests
    • Parallel requests can be used for fetching other resources, CSS, images, etc
  • You can load in more than just javascript modules
    • CSS, templates, etc.
